Project Description

SimuCrisis is an educational simulation platform developed with Unity to demonstrate pandemic disaster management. The system models virus spread and response strategies within a virtual population using historical data. Users act as policymakers to implement actions such as lockdowns, vaccinations, and healthcare investments.

Through interactive visual analytics including charts, heatmaps, and reports, the platform illustrates how different strategies influence infection rates, recovery outcomes, and resource use. The backend manages simulation results and pandemic datasets, while the Unity interface provides visualization and real-time feedback.

SimuCrisis targets educators, students, and researchers, offering a safe environment to explore crisis decision-making. By combining data-driven modeling with visualization, the project enhances understanding of pandemic dynamics and supports educational and institutional preparedness.
